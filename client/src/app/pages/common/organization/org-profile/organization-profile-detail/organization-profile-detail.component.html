የፕላንና ልማት ቢሮ ዓላማዎች

ራዕይ

በ2022 ዓ.ም ኢኮኖሚውን በረጅም ጊዜ ዕይታ በመምራት፣ዜጎችን እኩል የልማት ተሳታፊና ፍትሀዊ ተጠቃሚ በማድረግ መሠረታዊ የኢኮኖሚ ለውጥና ዘላቂ ልማት ለማረጋገጥና ድህነትን ለመቀነስ የልህቀት ማዕከል መሆን፡፡

ተልዕኮ

የክልሉን ኢኮኖሚ ሳይንሳዊ በሆነ ጥበብ በረጅም ጊዜ ዕይታ መምራት፣ ክልላዊ የልማት ቅደም ተከተል በማውጣት ውስን የማስፈፀሚያ ሀብት ውጤታማ በሆኑ የልማት ሥራዎች ላይ ተመድቦ ሥራ ላይ እንዲውል ማድረግ፣ የክልሉን ህዝብ በልማት እኩል ተሳታፊና ፍትሀዊ ተጠቃሚ የሚያደርግ ፈጣን መዋቅራዊ ለውጥ ማምጣት፣ ዘላቂ ልማት ማረጋገጥ እና ድህነትን ትርጉም ባለው ደረጃ መቀነስ፡፡

  • ሁሉም ፈጻሚዎች ከተግባራቸው አኳያ የራሳቸውን የአገልግሎት አሰጣጥ ስታንዳርድ እንዲያዘጋጁና በተግባር እንዲያውሉ ማገዝ፣
  • ተግባራት በተቀመጠላቸው ጊዜ መሰረት እንዲፈጸሙ ጥብቅ የስራ ዲሲፕሊንና የክትትልና ድጋፍ ስርዓትን መከተል፣
  • ለዉሳኔ ሰጭ አካላት እና ለአመራሮች ለስነ-ህዝብ ጉዳዮች ትኩረት ሰጥተው እንዲሰሩ ለማድረግ በክልሉ የበላይ አመራሮች በኩል አቅጣጫ እንዲሰጥ ማድረግ፤
  • በዕቅድ የተያዙ የካፒታል ፕሮጀክቶች ዕቀድ በማዘጋጀትም ሆነ በመተግበር ሂደት የሚያጋጥሙ ችግሮችንና ክፍተቶችን በጥናት በመለዬት ክፍተቶችን መሙላት፣
  • በክልሉ ውስጥ ወጥ የሆነ አሰራር ከመዘርጋት አንጻር አዳዲስ የካፒታል ፕሮጀክቶች አስተዳደር አዋጅ ማጸደቅ፣ የስልጠና ማንዋሎችን ማዘጋጀት፣ የአሰራር ማዕቀፎችን፣ ስርዓቶችንና መመሪያዎችን እንዲዘጋጁ በማድረግ ወደ ስራ እንዲገቡ ማድረግ፤
  • የክልል ኢኮኖሚ አካውንት አጠቃላይ አሰራርና አጠቃቀም በተመለከተ የመረጃ አቅራቢዎችና ተጠቃሚዎች የውይይት መድረኮች (Producer - User Forum) በማዘጋጀት ግብረ -መልሶችን በማሰባሰብ የአሰራር ሂደቶችን ለማሻሻል በግብዓትነት የመጠቀም አቅጣጫን፤
